# 파이터 스크롤 게임 구현 계획

## 프로젝트 개요
- 세로 스크롤 슈팅 게임
- 플레이어 전투기 조종, 상하좌우 이동
- 3번 피격 시 게임 오버
- 스테이지별 보스 전투

## 페이즈별 구현 순서

### Phase 1: 기본 시스템 (플레이어)
- 플레이어 전투기 이동 시스템
- 기본 공격 시스템
- 피격 및 생명 시스템
- 필사기 시스템 (3회, 3초 무적)

### Phase 2: 적 및 보스 시스템
- 일반 적 생성 및 이동 (화면 아래로)
- 적 HP 및 점수 시스템
- 보스 전투 시스템 (고정 위치)
- 보스 패턴 및 공격

### Phase 3: 아이템 시스템
- 적 처치 시 아이템 드롭
- 아이템 이동 (상하좌우)
- 공격력/연사 속도 업그레이드 아이템
- 필사기 충전 아이템

### Phase 4: 업그레이드 시스템
- 공격력 최대 6단계 강화
- 연사 속도 최대 6단계 강화
- 필사기 최대 3회 충전
- 전투기별 성능 차이

### Phase 5: 게임 플로우
- 스테이지 클리어 시스템
- 다음 스테이지 이동
- 게임 오버/클리어 화면
- 점수 시스템

## 개발 워크플로우
1. 리서치 (의존성 검토)
2. 구현계획 (Task List + 테스트 계획)
3. 구현 (TDD 방식)
4. 유저검수
