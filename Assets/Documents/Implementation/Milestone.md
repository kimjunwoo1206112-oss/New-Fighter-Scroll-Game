# 마일스톤 문서

## Phase별 마일스톤

---

## Milestone 1: 기본 플레이어 시스템
**Phase**: Phase 1  
**목표**: 플레이어 전투기 이동, 공격, 피격, 필사기 시스템 완성

### 완료 조건
- [ ] 플레이어가 화면 내에서 상하좌우 이동 가능
- [ ] 기본 공격 미사일 발사
- [ ] 피격 시 생명 차감 (3회)
- [ ] 필사기 사용 시 3초 무적 + 전체 데미지
- **개발자 검수**: 이동/공격/피격/필사기 정상 동작 확인
