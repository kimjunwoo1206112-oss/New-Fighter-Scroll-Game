# Phase 1: 기본 플레이어 시스템

## 개요
플레이어 전투기의 이동, 공격, 피격, 필사기 시스템 구현

## 할 일 목록

### 1. 플레이어 이동 시스템
- [ ] 상하좌우 이동 구현
- [ ] 화면 경계 제한
- [ ] 이동 속도 설정

### 2. 기본 공격 시스템
- [ ] 미사일 발사 로직
- [ ] 연사 속도 관리
- [ ] 공격력 기반 데미지

### 3. 피격 및 생명 시스템
- [ ] 피격 감지 (Collider)
- [ ] 생명 3회 관리
- [ ] 게임 오버 처리
- [ ] 피격 시 무적 시간 (短暂的)

### 4. 필사기 시스템
- [ ] 필사기 충전 관리 (최대 3)
- [ ] 필사기 사용 로직
- [ ] 무적 상태 3초 구현
- [ ] 전체 적 데미지 처리

### 5. 테스트 검증
- [ ] 이동 테스트
- [ ] 공격 테스트
- [ ] 피격 테스트
- [ ] 필사기 테스트

## 의존성
- Unity Input System
- Physics2D

## 완료 조건
- 플레이어가 화면 내에서 자유롭게 이동 가능
- 미사일이 정상적으로 발사되고 적에게 데미지 주기
- 피격 시 생명 차감 및 게임 오버
- 필사기 사용 시 무적 상태 및 전체 데미지
